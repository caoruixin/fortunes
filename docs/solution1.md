
# 道路检查井井周差异沉降“AI诊断 + 快速材料 + 微创加固”一体化解决方案

## 一、方案总目标

本项目不建议只做“材料研发”或“单一注浆工艺”，而应形成一套完整闭环：

**快速发现病害 → AI量化诊断 → 分级处置 → 微创加固 → 2小时开放交通 → 数字化验收与复盘。**

目标是把检查井维修从传统“看到坏了再开挖”升级为“可检测、可量化、可预测、可复制”的标准化养护产品。

现行道路养护思路本身也强调定期检测、快速处治、采用新技术新材料新工艺，并将检查井框与路面高差、井盖完好、路基沉陷、空洞等纳入巡查和特殊检测内容。([zjw.sh.gov.cn][1]) 市政检查井维修相关规程也明确，出现井盖沉陷、震响、井周破损等情况应维修加固，修补前井座周边结构层应夯填密实，快速恢复交通时宜采用快速修复材料并尽量减少占路时间和面积。([zjt.hubei.gov.cn][2])

# 二、核心判断：井周病害不是单一“井盖问题”，而是结构协同失效

井周差异沉降通常由四类问题叠加造成：

1. **井圈/井座体系刚度突变**
   检查井是刚性构筑物，道路结构层是相对柔性体系，车辆荷载反复作用后，井周容易形成应力集中。

2. **井周回填和基层压实不足**
   传统施工中井周狭窄区域难以充分碾压，形成松散层、空隙、水损和后期沉陷。

3. **井盖—井座—承压圈连接不可靠**
   井盖间隙、橡胶垫老化、限位失效、承压圈松动会造成跳盖、异响和冲击荷载放大。排水检查井技术规范也强调井盖不应直接坐落在井筒上，应设置座圈或承压圈；机动车道井盖承载等级、接触面、防震响橡胶圈、防位移、防噪声等均需控制。([amr.sz.gov.cn][3])

4. **水损、管道渗漏或地下空洞诱发持续发展**
   若只是表面修补，不处理深层空洞、松散基层或管道渗漏，短期内很容易复发。

因此，方案要同时解决三个问题：**看得见、补得牢、修得快。**

# 四、技术方案一：无损检测与AI精准诊断

## 4.1 检测组合

建议采用“多源检测”而不是单一探地雷达：

**主检测：三维探地雷达 / 多频探地雷达**

用于识别井周0—1.5米范围内的：

* 深层空洞；
* 基层松散；
* 沥青面层与基层脱空；
* 水损区；
* 管线周边异常；
* 井筒外侧回填不密实区。

相关技术规范也提出，地面坍塌隐患检测可根据对象、深度、致塌因素选择二维或三维探地雷达，管涵密集区域宜结合管道探地雷达；探地雷达识别出的空洞、脱空等隐患体应通过钻探、挖探、触探或其他方法复核验证。

**辅助检测：**

* 3D激光/结构光扫描：测井盖高程、井周沉陷盆、三米直尺平整度预测；
* 轻型落锤弯沉或便携弯沉仪：判断井周承载能力；
* 声振检测：识别井盖跳动、空响和接触不良；
* 管道CCTV或井内快速影像：排查管道接口渗漏、井筒破损；
* 现场含水率/温度检测：为材料配方和开放交通时间提供依据。

## 4.2 AI诊断模型

建立“井周病害AI识别模型”，输入数据包括：

* 探地雷达B-scan / C-scan图像；
* 井周路面照片；
* 高程差、平整度、裂缝分布；
* 声振信号；
* 历史维修记录；
* 交通等级、道路结构、井型、管线类型。

模型输出：

1. **异常类型识别**
   空洞、松散、脱空、水损、井座松动、井筒缺陷。

2. **异常范围量化**
   输出平面范围、深度范围、体积估算和置信度。

3. **风险等级分级**
   A级：轻微表层病害；
   B级：浅层脱空或井座松动；
   C级：基层松散/局部空洞；
   D级：深层空洞、管道渗漏或结构性破坏。

4. **维修策略推荐**
   自动建议“仅井盖锁固”“微孔注浆”“井周环切局部置换”“局部开挖重建”等工法。

## 4.3 准确率实现路径

企业提出“病害识别准确率90%以上”，建议这样落地：

* 第一阶段采集不少于300座检查井数据；
* 其中20%—30%做钻孔、取芯、局部开挖或CCTV复核；
* 建立“探地雷达异常—实际病害”标注库；
* 用AI模型进行盲测；
* 准确率不只看“有没有病害”，还要看：

  * 病害类型识别准确率；
  * 平面定位误差；
  * 深度判断误差；
  * 误报率；
  * 漏报率。

建议验收指标：

| 指标       | 建议目标       |
| -------- | ---------- |
| 单井检测时间   | 5—10分钟     |
| 病害识别准确率  | ≥90%       |
| 平面定位误差   | ≤10cm—20cm |
| 深度分层误差   | ≤10cm—15cm |
| 严重病害漏报率  | ≤5%        |
| 自动报告生成时间 | ≤3分钟       |

# 五、技术方案二：快速修复材料体系

建议研发两类核心材料，而不是单一材料包打天下。

## 材料A：井周快硬低收缩水泥基复合材料

用于井座调整、环形槽局部置换、井周承压过渡层。

### 目标性能

| 性能      | 建议目标           |
| ------- | -------------- |
| 初凝时间    | 10—20分钟可调      |
| 2小时抗压强度 | ≥20—30MPa      |
| 28天抗压强度 | ≥50MPa         |
| 抗折强度    | ≥6MPa          |
| 粘结强度    | ≥1.5MPa        |
| 收缩率     | 低收缩，优先控制早期塑性收缩 |
| 抗冲击     | 满足重载反复碾压       |
| 施工适应性   | 夜间、低温、潮湿基层可施工  |

### 配方方向

建议采用：

* 硫铝酸盐水泥/快硬水泥体系；
* 聚合物乳液或可再分散乳胶粉；
* 微硅粉、矿物掺合料；
* 纤维增强材料；
* 收缩补偿组分；
* 高性能减水剂；
* 沥青界面兼容粘结剂。

该材料重点解决“2小时开放交通”和“井座周边承压过渡层不再松散”。

## 材料B：可控扩散型微创注浆材料

用于井周深层空洞填充、基层松散加固和局部脱空处理。

建议形成两个子版本：

### B1：无机微细水泥基注浆料

适合：

* 基层松散；
* 回填不密实；
* 需要提高整体刚度和承载力的区域。

特点：

* 成本低；
* 与土体、基层兼容性较好；
* 强度发展稳定；
* 适合批量市政养护。

### B2：有机/无机复合快速注浆料

适合：

* 小空洞；
* 局部脱空；
* 潮湿环境；
* 需要快速形成支撑的病害。

特点：

* 低黏度；
* 快速固化；
* 轻微可控膨胀；
* 能快速填充复杂空隙。

注意：膨胀材料必须严控膨胀倍率和注浆压力，否则可能造成路面顶升、井筒挤压或浆液串入管道。

# 六、技术方案三：微创加固工艺

## 6.1 分级处置策略

| 病害等级 | 病害表现               | 推荐工法                       |
| ---- | ------------------ | -------------------------- |
| A级   | 井盖轻微异响、井盖间隙大、无深层异常 | 更换胶圈、限位锁固、井盖调平             |
| B级   | 井座松动、浅层脱空、轻微沉陷     | 井座锁固 + 环形快修材料              |
| C级   | 井周基层松散、局部空洞、沉陷明显   | 微孔注浆 + 局部环切修复              |
| D级   | 深层空洞、管道渗漏、井筒破损     | CCTV复核 + 局部开挖/管网修复，不建议单纯注浆 |

新建或改造导则中也提出，检查井基础地基强度、井周回填材料、分层压实、防沉降井盖、卸载板或分离式结构等都应根据井筒高度和道路条件统筹考虑。 这意味着本项目的微创工法应优先用于“井周结构尚可、局部失稳”的场景；对于结构性破坏，仍需局部重建。

## 6.2 标准施工流程

**步骤1：现场快速检测**

* 扫描井周1.5m—2.0m范围；
* 生成病害热力图；
* 标记注浆孔位或环切范围；
* 形成施工处置单。

**步骤2：微孔布孔**

建议采用“环向 + 放射状”布孔：

* 距井座边缘30cm—60cm设第一圈孔；
* 视病害范围设置第二圈孔；
* 孔径控制在小孔径范围；
* 孔距根据空洞范围、材料扩散半径和路面结构确定。

**步骤3：低压分级注浆**

施工控制重点：

* 小流量、低压力、分段注浆；
* 实时记录压力、流量、注浆量；
* 同步监测井盖和路面微小抬升；
* 发现串浆、冒浆、异常压力突升立即停止；
* 注浆后进行雷达或声振复检。

**步骤4：井座/井圈锁固与调平**

针对噪音源，不能只补路面，必须处理“盖—座—圈”体系：

* 更换或加装防震响整体橡胶圈；
* 加装限位、防位移、防跳动结构；
* 修复或重建井座承压面；
* 井盖与路面标高控制在设计允许范围内；
* 必要时采用自调式防沉降井盖或承压过渡环。

**步骤5：表层快速恢复**

* 环切范围尽量控制在病害实际范围内；
* 使用快硬低收缩复合材料恢复井周承压层；
* 上部与沥青面层之间设置粘结过渡层；
* 修复后2小时开放交通；
* 用三米直尺复测，目标控制在3mm以内。

# 八、AI系统与业务平台设计

建议企业最终看到的不只是施工工法，而是一个可运营的平台。

## 8.1 平台名称建议

**“井周病害AI诊修一体化平台”**

## 8.2 功能模块

1. **井盖资产地图**

   * 显示每座检查井位置、类型、权属、道路等级、维修历史。

2. **AI诊断中心**

   * 上传雷达数据、照片、声振数据；
   * 自动识别病害；
   * 输出风险等级和维修建议。

3. **施工决策助手**

   * 自动推荐材料类型；
   * 推荐注浆孔位；
   * 推荐注浆量范围；
   * 推荐开放交通时间。

4. **施工过程监管**

   * 实时记录注浆压力、流量、用料量；
   * 异常自动报警；
   * 支持远程质检。

5. **验收与复盘**

   * 自动生成验收报告；
   * 对比修复前后平整度、声振、雷达图；
   * 形成复发风险预测。

## 8.3 AI Agent能力

可以设计为5个Agent协同：

| Agent     | 作用                   |
| --------- | -------------------- |
| 检测Agent   | 指导现场人员采集雷达、照片、声振数据   |
| 诊断Agent   | 自动识别空洞、松散、脱空、水损、井座松动 |
| 工法Agent   | 根据病害等级推荐工艺和材料        |
| 施工质控Agent | 监测注浆压力、流量、开放交通时间     |
| 养护决策Agent | 根据复发风险生成年度养护计划       |

这会明显体现团队的AI技术优势，不只是“做施工”，而是帮企业建立可复制的智能养护能力。

# 十、实施计划

## 阶段一：调研与基线测试，2—3周

工作内容：

* 选择30—50座典型病害检查井；
* 收集道路结构、井型、交通量、历史维修记录；
* 进行雷达、声振、平整度、井内影像检测；
* 选取部分点位钻孔或局部开挖验证。

交付物：

* 病害分型报告；
* 检测数据样本库；
* 典型病害机理分析；
* 初版诊断指标体系。

## 阶段二：材料研发与室内试验，1.5—2个月

工作内容：

* 研发快硬低收缩水泥基材料；
* 研发微创注浆材料；
* 测试凝结时间、2小时强度、粘结强度、收缩、疲劳、水稳定性；
* 与沥青混凝土、井座、基层材料进行界面适配试验。

交付物：

* 1—2种材料配方；
* 材料性能检测报告；
* 施工窗口期建议；
* 不同温度环境下开放交通曲线。

## 阶段三：工艺与装备样机，2—3个月

工作内容：

* 完成注浆孔位设计；
* 完成小型注浆装备选型或改造；
* 建立压力—流量—注浆量控制逻辑；
* 完成现场小试。

交付物：

* 微创加固施工工法；
* 装备样机或装备组合清单；
* 施工质量控制表；
* 安全与交通组织方案。

## 阶段四：示范工程，3—6个月

建议选择100座左右检查井开展试点，覆盖：

* 主干路；
* 次干路；
* 公交线路；
* 重载车辆道路；
* 雨污水检查井；
* 通信/电力类检查井。

交付物：

* 示范工程总结；
* AI识别准确率验证报告；
* 修复后平整度、噪音、沉降复测报告；
* 成本与工期对比报告；
* 企业内部技术规程草案。

# 十一、关键验收指标

| 类别  | 指标        | 目标          |
| --- | --------- | ----------- |
| 诊断  | 病害识别准确率   | ≥90%        |
| 诊断  | 单井检测时间    | 5—10分钟      |
| 材料  | 开放交通时间    | ≤2小时        |
| 材料  | 早期强度      | 满足车辆开放要求    |
| 工艺  | 建筑垃圾减少    | ≥50%        |
| 工艺  | 非开挖/少开挖比例 | ≥70%        |
| 平整度 | 三米直尺检测    | ≤3mm        |
| 噪音  | 井盖跳动异响    | 验收时无明显跳动、空响 |
| 寿命  | 修复寿命      | 不低于周边原状道路   |
| 数字化 | 一井一档覆盖率   | 100%        |

“彻底消除噪音”在对外表达时建议改成更工程化的表述：**“维修验收状态下无明显跳动、空响和异常冲击声，并建立复检机制。”** 这样更严谨，也更容易验收。

# 十二、经济可行性分析

## 12.1 对企业的价值

相比传统大开挖维修，本方案的价值主要来自：

1. **减少开挖面积**

   * 降低人工、机械、材料和外运成本；
   * 减少建筑垃圾；
   * 降低夜间施工扰民。

2. **缩短占道时间**

   * 适合城市道路夜间“天窗期”；
   * 降低交通组织成本；
   * 提高道路养护响应能力。

3. **降低复发率**

   * 不是只修表面，而是处理空洞、松散层、井座松动等根因；
   * 形成可追溯的维修档案。

4. **形成新的业务产品**

   * 从“养护施工队”升级为“智能养护服务商”；
   * 可向市政道路、园区道路、机场港区、物流园、工业园区复制推广。

## 12.2 商业模式建议

可设计三种收费模式：

### 模式一：单井诊断收费

适合前期切入。

* 按检查井数量收费；
* 输出检测报告和维修建议；
* 企业容易接受，决策门槛低。

### 模式二：诊断 + 修复一体化收费

适合示范工程。

* 按病害等级计价；
* A/B/C/D级对应不同单价；
* 体现技术含量和施工价值。

### 模式三：年度智能养护服务

适合长期合作。

* 对一定片区检查井进行年度巡检；
* 建立风险地图；
* 对高风险井提前处置；
* 按年度服务费 + 单项维修费收费。

# 十三、主要风险与控制措施

| 风险        | 表现             | 控制措施                  |
| --------- | -------------- | --------------------- |
| 探地雷达误判    | 管线、钢筋、水分影响雷达图像 | 多频雷达 + 声振 + 局部钻孔验证    |
| 注浆串浆      | 浆液进入管道或空隙外溢    | 先做CCTV检查，低压分级注浆，异常停浆  |
| 路面顶升      | 膨胀材料或压力过大      | 设置位移监测，控制压力和注浆量       |
| 材料早强不足    | 2小时不能开放交通      | 建立温度—强度—开放交通曲线        |
| 低温/雨天施工失败 | 凝结慢、粘结差        | 设置冬季型、潮湿型配方           |
| 井盖噪音复发    | 只补路面，未处理盖座间隙   | 必须同步处理胶圈、限位、锁固和承压面    |
| 业主验收争议    | “彻底消除”难量化      | 用平整度、声振、跳动位移、复检数据量化验收 |
| 工法适用边界不清  | 深层管道渗漏仍用微创注浆   | 建立D级病害退出机制，转局部开挖或管网修复 |

# 十四、建议对企业呈现的最终方案亮点

对客户汇报时，建议突出五句话：

1. **不是简单修井盖，而是解决井周差异沉降的根因。**
2. **不是大开挖，而是基于AI诊断的精准微创修复。**
3. **不是凭经验施工，而是每座井都有病害热力图、注浆数据和验收档案。**
4. **不是一次性项目，而是可形成企业标准工法和年度养护产品。**
5. **不是单点技术，而是检测、材料、装备、施工、平台一体化解决方案。**
